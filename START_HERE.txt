═══════════════════════════════════════════════════════════════════════════════
APPROVAL WORKFLOW EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

CAPEX Purchase:
  
  Amount < 1,00,000 (1 Lakh):
    Requester → Manager → Approved

  Amount 1,00,000 - 10,00,000 (1-10 Lakhs):
    Requester → Manager → Director → Approved

  Amount > 10,00,000 (>10 Lakhs):
    Requester → Manager → Director → CFO → Approved
    (With IPC Committee if required)

OPEX Purchase:

  Amount < 50,000:
    Requester → Manager → Approved

  Amount 50,000 - 5,00,000:
    Requester → Manager → Finance_Head → Approved

  Amount > 5,00,000:
    Requester → Manager → Finance_Head → CFO → Approved
